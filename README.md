# Snake Game in Python
# gsnake
## Nokia classic snake game remake

### Technologies used
- PyGame

### Requires
- Python 3
- PyGame 2.5.2

## How to play
- Install via pip
```shell
python3 -m pip install gsnake
```
- Download it from the [PyPI Link](https://pypi.org/project/gsnake/)
- Run the game using Python
```shell
python3 -m gsnake
```
